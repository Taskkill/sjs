# sjs
Simple compiler from Symbolic JavaScript (Lisp-like language) to JavaScript. Created as semestral project.

## to run:
- install sbt
- `$ sbt`
- `compile`
- `run code.sjs`

## to run compiled program:
- install node
- `$ node a.js`

You can change content in the `code.js` - file is inside `src/main/resources`.
